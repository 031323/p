# पर्वयन्त्रम्
सङ्गणकेविधिनिवेशाययन्त्रमस्त्येतत्।

## उपभोगविधिः
एनम् Git निधानंस्वकीयेसङ्गणकेवाहरेत्।
ततः `emscripten` नामसञ्चयम्प्रविश्यतत्र `make` कुर्यात्।
```
cd emscripten
make
```
ततः `पर्वयन्त्रम्.html` नामसञ्चिकाञ्जालगवेषकउद्घाट्योपभुञ्जीत।
